# Assets

Place the game's media here (they are loaded by relative path from the project root):

```
assets/
├── images/     # all .png / .jpg / .jpeg sprites, maps and menus
└── musique/    # bgsound.wav
```

Copy the `images/` and `musique/` folders from your original project into this
`assets/` directory. File names are **case-sensitive** on Linux/macOS — keep them
exactly as referenced in the code (e.g. `Soigneuse1.png`, `Regles.png`, `Map7.jpeg`).
